# Contributing to Maestro Framework

## Code of Conduct
* Be respectful and considerate of others.
* Avoid spamming or self-promotion.
## Contribution Guidelines
1. Fork the repository: `git fork https://github.com/maestro-framework/maestro.git`
2. Create a new branch: `git branch <branch_name>`
3. Make changes and commit: `git commit -m '<commit_message>'`
4. Push changes: `git push origin <branch_name>`
5. Create a pull request: `git pull-request <branch_name>`
